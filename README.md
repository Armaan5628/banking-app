Royal Bank of Canada — Flutter Mobile Banking App

A modern Flutter-based banking application that simulates a real digital banking experience.
The app showcases a clean, elegant design built using Flutter and Dart — featuring multiple screens such as the Welcome Page, Account List, and Transaction Details, complete with dynamic JSON data, masked account numbers, and sorted transaction history.

 Features

 Welcome Screen with elegant gradient background and dynamic date display
 Account List Screen showing account type, balance, and masked account numbers (only last 4 digits visible)
 Transaction Details Screen with formatted dates and color-coded deposits/withdrawals
 Transactions sorted from newest to oldest
 Premium Royal Blue UI theme matching real-bank interfaces
 Privacy-friendly display (asterisk masking for account numbers)
 Fully responsive across mobile and web
